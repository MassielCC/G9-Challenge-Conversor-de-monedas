## Challenge: Conversor de monedas en Java
### Descripción:
Este proyecto es una aplicación de consola desarrollada en Java que permite convertir montos entre distintas monedas, utilizando la ExchangeRate API para obtener las tasas de cambio en tiempo real.

### Funcionalidades
* Menú para interacción con usuario
* Convertir de dólar a pesos argentinos, reales brasileños y pesos colombianos y viceversa.
* Consumo de una API para obtener las tasas actuales, se utilizó el Pair Conversion 
* Uso de la librería Gson para leer la respuesta json obtenida de la API

### Requisitos para usar el proyecto
* Java 17 o superior
* Paquete Gson 2.10.1 o superior
* Api Key para usar ExchangeRate-API
